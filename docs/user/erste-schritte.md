# Erste Schritte

## Anmelden

Die Anmeldung bei ki@schule erfolgt über Ihr Schulkonto. Klicken Sie auf der Startseite auf **„Anmelden"** — Sie werden zur gewohnten IServ-Anmeldung weitergeleitet. Nach erfolgreicher Anmeldung kehren Sie automatisch zur Plattform zurück.

Ein separater Account ist nicht notwendig. Wer ein Schulkonto hat, kann sofort loslegen.

## Die Oberfläche auf einen Blick

Nach der Anmeldung sehen Sie die Hauptansicht der Plattform:

**Seitenleiste (links)**
- **Neuer Chat** — startet ein leeres Gespräch
- **Verlauf** — zeigt Ihre früheren Gespräche
- **Assistenten** — öffnet die Liste verfügbarer Assistenten

**Hauptbereich (Mitte)**
- Der Chat-Bereich, in dem Gespräche stattfinden

**User-Menü (oben rechts, Ihr Name)**
- **Profil** — Budget und persönliche Einstellungen
- **Über diese Software** — Version und Lizenzinformationen
- **Abmelden** — sicher ausloggen

## Ersten Chat starten

1. Klicken Sie in der Seitenleiste auf **„Neuer Chat"**.
2. Tippen Sie Ihre Frage oder Aufgabe in das Eingabefeld am unteren Rand.
3. Senden Sie die Nachricht mit **Enter** oder dem Senden-Button.
4. Die KI antwortet direkt im Chatfenster. Sie können die Antwort lesen und
   im gleichen Gespräch weiter fragen — die KI kennt den bisherigen Gesprächsverlauf.

Ein Gespräch kann so lange fortgeführt werden, wie Sie möchten. Wenn Sie ein neues, unabhängiges Thema beginnen, empfiehlt es sich, einen **neuen Chat** zu starten — so bleibt jedes Gespräch übersichtlich.
