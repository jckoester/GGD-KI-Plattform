# Assistenten

## Was ist ein Assistent?

Ein Assistent ist ein vorbereiteter Chat mit einer bestimmten Rolle oder Aufgabe. Hinter jedem Assistenten steckt eine Anleitung, die der KI erklärt, wie sie sich verhalten soll — zum Beispiel als Nachhilfe-Tutor, als Korrektur-Assistent oder als kreativer Schreibpartner.

Das spart Ihnen das wiederholte Erklären: Sie öffnen den Assistenten und können sofort loslegen, ohne erst beschreiben zu müssen, was die KI tun soll.

Assistenten werden von Lehrkräften oder Administratoren angelegt und für bestimmte Gruppen oder alle Nutzerinnen und Nutzer freigegeben.

## Assistenten finden und starten

1. Klicken Sie in der Seitenleiste auf **„Assistenten"**.
2. Sie sehen eine Liste der verfügbaren Assistenten mit einer kurzen Beschreibung.
3. Klicken Sie auf einen Assistenten, um einen neuen Chat mit ihm zu starten.

Der Assistent ist für die gesamte Dauer des Gesprächs aktiv. Wenn Sie einen anderen Assistenten nutzen möchten, starten Sie einen neuen Chat.

## Unterschied zum normalen Chat

| | Normaler Chat | Assistent |
|---|---|---|
| Rolle der KI | Allgemein | Vorgegeben (z. B. Tutor, Korrektor) |
| Modell | Frei wählbar | Ggf. vom Assistenten vorgegeben |
| Einstieg | Freie Frage | Thema ist bereits gesetzt |

Im Verlauf sind Assistenten-Gespräche mit einem Icon gekennzeichnet, damit Sie sie leicht von normalen Chats unterscheiden können.

## Eigene Assistenten erstellen

Das Anlegen von Assistenten ist Lehrkräften und Administratoren vorbehalten. Wenn Sie eine Idee für einen nützlichen Assistenten haben, sprechen Sie Ihre Lehrkraft oder den IT-Verantwortlichen an.
